## 📝 標準エディター対応 メニュー編集システム 詳細手順書

### 🎯 **完全な編集手順（MENU番号読み込み後）**

---

## **ステップ1: MENU番号で検索**

1. **管理画面アクセス**
   - WordPressの管理画面にログイン
   - 左メニューの「投稿」→「✨ 標準エディター」をクリック

2. **MENU番号入力**
   - 「MENU番号」欄に編集したい番号を入力（例：`01-02-01`）
   - 「🔍 検索・編集」ボタンをクリック

---

## **ステップ2: MENU投稿の確認**

### ✅ **投稿が見つかった場合**
画面に以下の情報が表示されます：
- ✅ MENU「01-02-01」が見つかりました
- タイトル: [投稿のタイトル]
- 投稿ID: [数字]
- 状態: publish/draft
- 最終更新: [日時]

### ❌ **投稿が見つからない場合**
- 「MENU番号が見つかりません」メッセージ
- 新規作成オプションが表示される

---

## **ステップ3: WordPress標準エディターで編集**

### 🔥 **重要：ここからが本番の編集作業**

1. **編集ボタンをクリック**
   ```
   📝 WordPress標準エディターで編集
   ```
   このボタンをクリックします

2. **WordPress標準編集画面が開く**
   - 新しいタブまたはウィンドウでWordPressの標準投稿編集画面が開きます
   - これは通常の投稿編集と全く同じ画面です

---

## **ステップ4: 標準エディターでの編集作業**

### 📝 **ビジュアルエディターで編集（推奨）**

1. **テキスト入力・編集**
   - 本文エリアでテキストを自由に入力・編集
   - 改行・段落も自由に設定可能

2. **フォントと文字装飾**
   ```
   ツールバーから選択可能：
   - B（太字）、I（斜体）、U（下線）
   - フォントサイズ変更
   - 文字色・背景色変更
   - 左寄せ・中央寄せ・右寄せ
   ```

3. **高度な装飾**
   ```
   - 箇条書きリスト（・）
   - 番号付きリスト（1.2.3.）
   - 表（テーブル）挿入
   - 画像挿入
   - リンク挿入
   - 引用ブロック
   ```

4. **ブロックエディター（Gutenberg）の場合**
   ```
   - + ボタンで新しいブロック追加
   - 段落、見出し、画像、表、リストなど
   - 各ブロックごとに装飾設定
   ```

---

## **ステップ5: 保存（最重要）**

### 💾 **確実な保存方法**

1. **下書き保存**
   ```
   画面右側の「下書きとして保存」ボタン
   → 一時的に保存（公開されない）
   ```

2. **プレビュー確認**
   ```
   「プレビュー」ボタンで表示確認
   → 装飾やレイアウトが正しく表示されるかチェック
   ```

3. **公開保存**
   ```
   「公開」または「更新」ボタン
   → 実際にサイトに反映される
   ```

---

## **ステップ6: 確認作業**

### 🔍 **保存後の確認**

1. **管理画面での確認**
   - 保存完了メッセージを確認
   - 「投稿を表示」リンクで公開ページを確認

2. **元の検索画面に戻る**
   - ブラウザの戻るボタンで標準エディター画面に戻る
   - 同じMENU番号で再検索して変更が反映されているか確認

3. **公開サイトでの確認**
   - 実際のMENU一覧ページで表示確認
   - フォント・色・レイアウトが正しく表示されているか確認

---

## **🎯 重要なポイント**

### ✅ **成功のコツ**
- WordPress標準の投稿編集と全く同じ操作
- ビジュアルエディターを使用（テキストエディターより簡単）
- 保存前に必ずプレビューで確認
- 編集中は頻繁に「下書き保存」をクリック

### ⚠️ **注意点**
- カスタムフィールド「menu_no」は変更しない
- 投稿タイトルは分かりやすい名前に設定
- 公開状態を「公開」に設定（非公開だと表示されない）

### 🆘 **トラブル時の対処**
- 保存されない場合：ブラウザを更新して再試行
- 装飾が消える場合：ビジュアルエディターを使用
- 表示されない場合：投稿状態を「公開」に確認

---

- [ ] フォント・色・装飾が適用されている
- [ ] 表・画像・リンクが正しく挿入されている
- [ ] プレビューで表示確認済み
- [ ] 「公開」または「更新」で保存完了
- [ ] 公開サイトで最終確認済み
